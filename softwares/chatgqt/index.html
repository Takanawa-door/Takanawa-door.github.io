    ChatGQT

    最新版本： Version 0.2 ；
    语言： 简体中文。

    ChatGQT 是新一代复读机，智障般地匹配数据库并且给出回答。

    • 自主思考。
    • 完善数据库。

    ChatGQT V0.2 发布¶

    ChatGQT V0.2 基于 动态规划。灵感来自某考试最后一道题的完善程序的编辑距离。

    提升¶

    1. 更加 智慧。动态规划算法相比于 ChatGQT V0.1 使用的 Low 的字符匹配（基于桶），更加智慧。动态规划可以更好算出预设问题与输入的相似度！
    2. 更好的 可移植性。对不同操作系统，移植方便而简单。
    3. 支持 自定义语料。只需要调用 ChatGQT 时加入语料文件（文本文件）就可以让 ChatGQT 方便问答！一份 ChatGQT ，百万语料百万问答。
    4. 支持 汉字。ChatGQT V0.1 不支持汉字，随着算法的改进， ChatGQT 已经可以识别汉字了！

    马上体验¶

    Github。下载 ChatGQT Version 2.0 ：https://wwwe.lanzouq.com/b01fjfkmd（密码:chatgqt ）。

    使用方法¶

    1. 新建文本文件 yuliao.txt。

    2. 向文件中写入如下格式的内容：

      "Question"

问题。

"Answer"

解答。

...

"End"

    3. 运行 chatgqt.exe 6000 yuliao.txt。其中 6000 指只有问题与输入相似度达到 60% 才会输出答案。

      你也可以下载预设语料库进行问答。

    4. 调教你的 ChatGQT ！

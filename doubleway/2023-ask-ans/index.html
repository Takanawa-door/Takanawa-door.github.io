    1. 单选题¶

    1. 下列语句中可以修改 a 的成员 x 为 5 的是（ ）。

      struct {
    int x;
    double y;
    char z;
} a;

      A. a.x = 5
      B. a.y = 5
      C. a->x = 5
      D. a.x = 10

    2. 下列选项中说法错误的是（ ）。

      A. 左值的定义是可以出现在赋值语句等于号左边的变量。
      B. 5 是右值。
      C. delete 关键字可以释放运行时申请的内存。
      D. NULL 与 nullptr （ C++11 起）都用于指代空指针，但是有本质上的区别。

    3. std::filesystem （头文件）是 C++（ ）中新加入的标准库。

      A. 11
      B. 14
      C. 17
      D. 20

    4. 若有 std::vector vec ，则与 for(int i = 0, &c = vec[i]; i < vec.size() ; c = vec[++i])等效的是（ ）。

      A. for (int& item : vec)
      B. for (int item : vec)
      C. while (i != vec.size())
      D. while (i != vec.end())

    5. 下列关键字与#include 命令有相似作用的是（ ）。

      A. extern
      B. module
      C. using
      D. import

    6. 已知如下代码，则下列选项中添加到该段代码之后会导致未定义行为的是（ ）。

      int a = 10, b = 20, c[10];
int *p1, *p2 = nullptr, *p3 = new int;
delete p3;

      A. c[1] = 20;
      B. p2 = &a;
      C. delete p3;
      D. p1 = new int[10];

    7. 对于表达式 const int* const (*p)(const int, const int)解读正确是（ ）。

      A. 如果指针 p 指向具有实际意义的地址则语句 p(10, 10)是不合法的。
      B. p 指向一个函数，返回值是指针，返回的指针完全不可以被修改。
      C. const int* const 中的第一个 const 表示指针 p 不可以指向其它地址。
      D. 由第二个 const 可知， p 是一个不可以修改指向地址的指针。

    8. 一个名字的函数可以出现多种不同类型参数的调用，则称（ ）。

      A. 这个函数具有其它的重载。
      B. 这个函数是多态的。
      C. 这个函数是继承自其它函数的。
      D. 不可能出现这种函数。

    9. 下列关于虚函数说法正确的是（ ）。

      A. 虚函数不能在派生类中出现。
      B. 虚函数可以在派生类中重新定义。
      C. 如果一个父类具有虚函数，则其派生类必须实现该虚函数。
      D. 父类中虚函数必须具有实现。

    10. 下列说法错误的是（ ）。

      A. C++11 以后 auto 关键字用于自动推导类型。
      B. C++20 起才能使用范围库（ ranges ）。
      C. assert(x)中，如果 x 的值为假会在运行期抛出异常。
      D. static_assert(x)中，如果 x 的值为假会在运行期抛出异常。

      问题描述

      世界各地的奶牛将会到达当地的机场，前来参会并且吃草。

      具体地说，有 N 头奶牛到达了机场（\(1 \leq N \leq 10^5\)），其中奶牛 i 在时间 \(t_i\)（\(0 \leq t_i \leq 10^9\)）到达。Farmer John 安排了 M （\(1 \leq M \leq 10^5\)）辆大巴来机场接这些奶牛。每辆大巴可以乘坐 C 头奶牛（\(1 \leq C \leq N\)）。

      Farmer John 正在机场等待奶牛们到来，并且准备安排到达的奶牛们乘坐大巴。当最后一头乘坐某辆大巴的奶牛到达的时候，这辆大巴就可以发车了。Farmer John 想要做一个优秀的主办者，所以并不想让奶牛们在机场等待过长的时间。如果 Farmer John 合理地协调这些大巴，等待时间最长的奶牛等待的时间的最小值是 多少？一头奶牛的等待时间等于她的到达时间与她乘坐的大巴的发车时间之差。
